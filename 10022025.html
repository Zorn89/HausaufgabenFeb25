Fragen und Antworten zum Thema Backups

1. Was sind Backups?

Backups sind Kopien von Daten, die zu einem bestimmten Zeitpunkt gesichert werden. Sie dienen dazu, wichtige Informationen oder Dateien vor Verlust zu schützen, falls diese durch Systemfehler, Viruserkrankungen oder menschliche Fehler gelöscht oder beschädigt werden.

Ein Backup kann auf verschiedenen Medien gespeichert werden, wie zum Beispiel auf externen Festplatten, in der Cloud oder auf Cloud-Servern.

2. Warum sollte man Backups machen?

Backups sind eine der wichtigsten Sicherheitsvorkehrungen, um den Verlust wertvoller Daten zu verhindern. Ohne Backups können Daten unwiederbringlich verloren gehen, was zu ernsthaften Problemen für Unternehmen und Privatpersonen führen kann. Einige Gründe für Backups sind:

  • Schutz vor Datenverlust: Wenn ein Gerät oder eine Festplatte ausfällt, ermöglicht ein Backup die Wiederherstellung der Daten.
  • Wiederherstellung von Dateien: Backups ermöglichen es, einzelne Dateien oder ganze Systeme wiederherzustellen, die versehentlich gelöscht oder beschädigt wurden.
  • Schutz vor Viren und Ransomware: Wenn ein Virus das System infiziert, können Daten durch das Backup schnell wiederhergestellt werden.

Weitere Details zu Sicherheitsaspekten und Backup-Strategien finden Sie in diesem Artikel des Bundesamts für Sicherheit in der Informationstechnik.

Schau dir diesen Code für eine Datei backup.py an:

  1. Importiert die Module shutil und datetime: shutil: Ein Modul, das verwendet wird, um Dateisystemoperationen wie das Kopieren von Verzeichnissen zu ermöglichen. datetime: Ein Modul, das für die Arbeit mit Datums- und Zeitinformationen zuständig ist.
  2. Definiert den Quellordner: backup_source = "C:/Users/Benutzer/Dokumente": Der Quellordner, dessen Inhalt gesichert werden soll, ist der Ordner „Dokumente“ des Benutzers.
  3. den Zielordner für das Backup: backup_target = "D:/Backup_" + datetime.datetime.now().strftime("%Y%m%d%H%M"): Der Zielordner für das Backup wird mit einem Namen erstellt, der das aktuelle Datum und die Uhrzeit im Format YYYYMMDDHHMM enthält (z.B. Backup_202502101200).
  4. Kopiert den Quellordner in den Zielordner: shutil.copytree(backup_source, backup_target): Der gesamte Inhalt des Quellordners (backup_source) wird in den Zielordner (backup_target) kopiert, einschließlich aller Unterverzeichnisse und Dateien.
  5. Gibt eine Bestätigung aus: print(f"Backup erfolgreich: {backup_target}"): Nachdem das Backup abgeschlossen ist, gibt der Code eine Bestätigungsmeldung aus, die den Pfad des Zielordners anzeigt.
        import shutil
        import datetime
        
        backup_source = "C:/Users/Benutzer/Dokumente"
        backup_target = "D:/Backup_" + datetime.datetime.now().strftime("%Y%m%d%H%M")
        
        shutil.copytree(backup_source, backup_target)
        print(f"Backup erfolgreich: {backup_target}")

Gibt es einen Weg ein Script automatisch wiederholt auszuführen? Also bspw. einmal pro Tag um 2 Uhr? Kannst du dir erklären wie das automatische Backups ermöglicht?

Ja.

  • Windows Task Scheduler (Windows):Auf Windows-Systemen kannst du den Task Scheduler (Aufgabenplanung) verwenden, um ein Skript zu einer bestimmten Zeit automatisch auszuführen. Du kannst ihn so konfigurieren, dass das Backup-Skript täglich um 2 Uhr nachts ausgeführt wird.
  • Cron Jobs (Linux/macOS): Auf Linux- und macOS-Systemen kannst du den Cron-Job verwenden, der die Möglichkeit bietet, Skripte in festgelegten Intervallen auszuführen. Du kannst einen Cron-Job erstellen, der das Backup-Skript täglich um 2 Uhr ausführt.
